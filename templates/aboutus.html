MLU Calculator: About Us



How did we do this?

We utilized the Vosk API to convert speech to text, for those who choose to upload an audio file.
Then, we used the Morfessor module, from NTLK, to count the number of morphemes in transcript.
Flask served as microframework for this web application, and MUI CSS served as the front-end.
No database service is being utilized for this service. All files (if any) are deleted within 5 minutes.



Future Steps

We plan to use ffmpeg to allow users to submit a wider variety of audio files.
